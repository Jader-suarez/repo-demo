# Guía para Etiquetar Documentos Markdown

Esta guía proporciona ejemplos y buenas prácticas para etiquetar y dar formato a documentos Markdown.

---

## 1. Encabezados

Usa `#` para crear encabezados (hasta 6 niveles).

```markdown
# Título nivel 1
## Título nivel 2
### Título nivel 3
#### Título nivel 4
##### Título nivel 5
###### Título nivel 6
